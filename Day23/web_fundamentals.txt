# HTTP = Hyper Text Transfer Protocol ---> creates communication protocol between client and server 
HTTP request --> a reuest sent by client to server asking for some information 

HTTP response -->response sent by server to client 


HTTP methods 

retrieve data --- GET method 
send data ---- POST method 
update data --- PUT method 
delete data --- DELETE method
